# 组合地名生成器

一个基于Web的中文地名生成工具，支持自定义词库、锁定字符和批量生成等功能。

## 功能特性

- 🎯 **灵活的地名字数设置**：支持2-10字地名生成
- 📚 **多层级词库系统**：包含单字、双字词、三字词、四字词四个分类
- 🎨 **丰富的词库内容**：内置形容词、方位词、水体、地形、行政区划、动物、植物、矿产、建筑、道路等多个词库
- 🔒 **字符锁定功能**：可锁定特定位置的字符，实现半定制化生成
- 📝 **自定义字符输入**：支持直接输入自定义字符
- 🔄 **批量生成**：一次生成多个地名，数量可调节
- 💾 **生成历史记录**：保存历史生成结果，支持点击复制
- 📤 **词库导入导出**：支持词库的导入导出，方便备份和共享
- 🔧 **词库管理**：支持新增、重命名、删除词库
- 📱 **响应式设计**：适配不同屏幕尺寸

## 快速开始

1. **打开项目**：在浏览器中打开 `index.html` 文件
2. **设置地名字数**：在"字符设置"区域选择要生成的地名字数
3. **选择词库**：为每个部分选择要使用的词库
4. **自定义字符**：可选择锁定特定位置的字符，或直接输入自定义字符
5. **设置生成数量**：在"生成设置"区域设置要生成的地名数量
6. **生成地名**：点击"生成单个地名"或"生成多个地名"按钮
7. **复制结果**：点击"复制结果"按钮复制生成的地名

## 词库管理

### 内置词库

- **单字**：形容词、方位词、水体、地形、行政区划、动物、植物、矿产、建筑、道路、姓氏
- **双字词**：方位、地理、建筑
- **三字词**：地理、建筑
- **四字词**：地理、建筑

### 自定义词库

1. **打开词库管理**：点击"打开词库管理"按钮
2. **新增词库**：点击"新增"按钮，输入词库名称并选择分类
3. **编辑词库**：在文本框中编辑词库内容，字符之间用空格或换行分隔
4. **保存更改**：点击"保存更改"按钮保存修改
5. **导入词库**：点击"导入"按钮，选择词库文件（.txt格式）
6. **导出词库**：点击"导出"按钮，选择要导出的词库

## 技术实现

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **样式设计**：使用Flexbox布局，支持响应式设计
- **数据存储**：使用浏览器本地内存存储词库数据
- **交互方式**：基于DOM操作和事件处理

## 使用技巧

1. **生成特色地名**：通过组合不同类型的词库，如"形容词+地形+行政区划"，可以生成更有特色的地名
2. **锁定关键字**：如果想生成包含特定字符的地名，可以使用锁定功能
3. **批量生成**：设置较大的生成数量，可以一次性获得多个选项
4. **词库扩展**：根据需要扩展词库，添加更多符合特定场景的词汇
